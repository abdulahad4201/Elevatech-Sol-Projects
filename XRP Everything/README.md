# XRP Everything – QA Testing Project

This repository showcases my Software Quality Assurance (QA) work on **XRP Everything**, a vehicle listing and promotion platform. The project includes web, mobile, and admin applications designed to connect vehicle dealers with buyers while providing XRP reference pricing and dealer promotional features.

---

## 📌 Project Overview

* **Project Name:** XRP Everything
* **Project Type:** Vehicle Listing & Promotion Platform
* **Testing Type:** Manual Testing & API Testing
* **Platforms:** Web Application, Mobile Application, Admin Portal

---

## 🧪 Testing Scope

During this project, I performed testing across multiple modules, including:

* User Authentication (Sign In, Sign Up, Forgot Password)
* Dealer & Buyer Management
* Vehicle Listings
* Search & Filter
* Favorites
* In-App Messaging
* Orders & Inquiry Flow
* Profile Management
* Admin Portal
* API Endpoints
* UI & Functional Testing

---

## 🐞 QA Activities

* Designed and executed test cases for different modules
* Identified and documented functional and UI defects
* Verified API responses using Postman
* Performed regression testing after bug fixes
* Validated input fields, error handling, and business workflows
* Tested application behavior across different user roles
