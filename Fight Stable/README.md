# Fight Stable – Mobile Application Testing

This repository showcases my QA testing work on **Fight Stable**, a mobile application developed for gym management. The platform consists of two separate mobile applications for **Coaches** and **Students**, providing features for membership management, communication, and subscription tracking.

---

## 📌 Project Overview

* **Project Name:** Fight Stable
* **Project Type:** Mobile Application
* **Testing Type:** Manual Testing
* **Platforms:** Coach App & Student App

---

## 🧪 Testing Scope

During this project, I performed testing across multiple modules, including:

* User Authentication (Sign In & Sign Up)
* Student Membership Plans
* Coach Dashboard
* Member Management
* Payment Status
* Chat Functionality
* Notifications
* Reminder Management
* Profile Management
* Settings
* Functional & UI Testing

---

## 🐞 QA Activities

* Executed functional testing for both Coach and Student applications
* Verified subscription plan workflows and feature accessibility
* Tested payment status and membership management features
* Validated real-time chat functionality between coaches and students
* Verified reminder and notification delivery
* Identified and reported functional, UI, and usability issues
* Performed regression testing after issue resolution
